Target Compiler

Generate the targets.mind file your AR demo needs.

STEP 01

Load the floor plan image

Loads floor_plan_1.webp from this folder and prepares it for tracking.

STEP 02

Compile to .mind format

This processes the image into MindAR's binary target format. Takes ~10–30 seconds.

STEP 03

Download targets.mind

Place this file in the same folder as index.html when you deploy.

STEP 04

Also download the printable target

You'll want a clean PNG of the floor plan to print or display. This is what users will scan.
